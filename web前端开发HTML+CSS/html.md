# Html标签

## <a>标签

> 功能：
>
> (1)超链接
>
> ```html
> <a href="baidu.com"
> ```
>
> 
>
> (2)锚点
>
> (3)打电话
>
> ```html
> <a href="tel:156928200983" style="font-size:12rem";></a>
> <a href="mailto:156928200983@163.com" style="font-size:12rem";></a>
> ```
>
> (4)协议限定符
>
> ```html
> <a href="javascript:while(1){alert}"></a>
> ```
